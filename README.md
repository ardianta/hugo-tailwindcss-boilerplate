## Requirement

- Hugo `extended` version.

## Install

```bash
npm install
```

## Run

You can run using Hugo server:

```bash
hugo server
```

or NPM:

```bash
npm start
```

## Build

To build for production, be sure to set the NODE_ENV variable:

```bash
NODE_ENV=production hugo
```

or just use this command:

```bash
npm run build
```
